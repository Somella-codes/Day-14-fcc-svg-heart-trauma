# Day-14-fcc-svg-heart-trauma

## Day 14: The Heart Icon That Broke Me 😭

**Task:** Draw a red heart with SVG. Sounds easy, right? 

**Reality:** FCC turned it into an hour-long boss fight.

I can beat my chest and say I wrote the code correctly. Triple-checked everything. 
FCC kept saying "Nope. Try again" like it was enjoying my pain 💀

The error message? Useless. 
My energy? Draining. 
My patience? Gone.

After an hour of debugging, console.log, and questioning my career choice...
I found it. 

**The bug:** I pressed Enter once. 
Just ONE line break inside the SVG path. 
That was the whole reason FCC refused to pass me.

Code really no dey respect effort abeg.

### **What FCC taught me today:**
1. **Syntax is king** - Your logic can be perfect, but one space can end you
2. **Error messages lie sometimes** - FCC said "d attribute missing" when the real issue was formatting  
3. **Small wins matter** - Seeing this red heart appear felt like I just deployed Facebook
4. **Document your pain** - If I can save 1 person from this SVG trauma, then the hour was worth it

If peradventure you get stuck on code too: Check your line breaks. 
Check them again. Then check once more. 

You’re not dumb. Building a webpage is not for the weak 💪
